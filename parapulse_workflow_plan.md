# ParaPulse: AI Community Issue Reporter

## One-Line Summary

**ParaPulse** is an AI-powered civic issue reporting tool that helps local residents and volunteers turn photos or short descriptions of community problems into structured, shareable, location-based action reports.

# 2. Problem Statement Judges Might Ask About

## Specific Problem

In many neighborhoods, people notice problems like garbage accumulation, broken streetlights, open drains, waterlogging, damaged footpaths, and unsafe public areas. However, reporting these issues is frustrating because residents often do not know:

- How to describe the issue formally.
- Which category the issue belongs to.
- How urgent the issue is.
- What information is missing.
- How to write a usable complaint.
- How to coordinate with other residents or volunteers.

As a result, issues remain scattered across chats and social media instead of becoming actionable community reports.

## Why Existing Solutions Are Not Enough

Many civic complaint portals exist, but they can be hard to use, city-specific, slow, or not designed for community-level volunteer coordination. ParaPulse is not trying to replace government systems. It helps communities create better reports before escalation.

## Real-World Example

A student sees waterlogging near a bus stop after rain. Instead of just posting “Road is flooded” in a group chat, they upload a photo to ParaPulse. The app identifies it as a waterlogging hazard, marks it as high severity, generates a formal complaint, pins it on a map, and provides a shareable message for local escalation.

# 3. Solution

## What ParaPulse Does

ParaPulse allows a user to:

1. Upload a photo or enter a short text description of a local civic issue.
2. Add or detect the location.
3. Use AI to classify the problem.
4. Generate a structured civic report.
5. Display the issue on a community map.
6. Create a copy-ready complaint message.
7. Track issue status.
8. Help volunteers identify urgent problems nearby.

## Core Output Generated by AI

For each issue, ParaPulse generates:

- **Issue category:** Garbage, waterlogging, road damage, streetlight, public safety, drainage, accessibility, etc.
- **Severity:** Low, medium, high, urgent.
- **Short title:** Example: “Severe waterlogging near bus stop.”
- **Detailed description:** Formal, clean summary.
- **Suggested authority:** Municipality, ward office, electricity department, sanitation team, traffic authority, etc.
- **Missing information:** Example: exact landmark, duration, contact number.
- **Complaint message:** Ready to copy/share.
- **Volunteer action suggestion:** What local volunteers can do next.

# 8. AI Component

## Why AI Is Meaningful Here

AI is not just a chatbot in ParaPulse. It performs real civic intelligence tasks:

1. Understands unstructured issue input.
2. Extracts civic problem type.
3. Estimates severity.
4. Converts casual text into formal complaint language.
5. Suggests the responsible authority.
6. Detects missing information.
7. Generates volunteer action guidance.

## AI Input

The model receives:

- User text description
- Optional image
- User-provided location
- Optional landmark

## AI Output Format

The AI should return structured JSON:

```json
{
  "title": "Severe waterlogging near bus stop",
  "category": "Waterlogging",
  "severity": "High",
  "summary": "Rainwater has accumulated near the bus stop, making pedestrian movement difficult and potentially unsafe.",
  "suggestedAuthority": "Municipal drainage department / local ward office",
  "missingInfo": ["Exact ward number", "How long the issue has persisted"],
  "complaintMessage": "Dear concerned authority, there is severe waterlogging near the bus stop at Salt Lake Sector V...",
  "volunteerAction": "Ask nearby residents to confirm duration and collect one more photo from a different angle."
}
```

## Prompt Strategy

The app should instruct the AI to behave like a civic issue analyst.

Prompt should ask the model to:

- Classify the issue.
- Assess urgency.
- Avoid exaggeration.
- Generate short, formal language.
- Return strict JSON.
- Mention uncertainty if image/text is unclear.

# 13. Questions Judges Might Ask

## What problem are you solving?

We are solving the gap between noticing a local civic issue and creating a structured, actionable report that can be shared, tracked, and escalated.

## Who is the target user?

Local residents, student volunteers, community organizers, NGOs, and resident welfare groups.

## Why does this need AI?

Most users submit messy, incomplete, or emotional descriptions. AI helps convert that unstructured input into a structured civic report with category, severity, authority suggestion, missing information, and complaint text.

## Is this just a complaint app?

No. The focus is not only complaint submission. ParaPulse is a community intelligence layer that helps residents and volunteers document, organize, prioritize, and escalate civic problems.

## How will you prevent fake reports?

For the MVP, we can show reports as community-submitted and allow status changes. Future versions can add duplicate detection, image metadata checks, volunteer verification, user trust scores, and multiple witness confirmations.

## What if AI gives the wrong authority or severity?

The AI output is presented as a suggestion, not a final legal decision. Users can edit reports before saving. Future versions can use city-specific authority mappings.

## What makes this community-focused?

It helps local people coordinate around shared problems. The map makes issues visible, the complaint generator makes escalation easier, and volunteers can prioritize urgent cases.

## Can this scale beyond Kolkata?

Yes. The core workflow works for any city. Local authority mappings, languages, and categories can be customized city by city.

## What is the business or sustainability model?

Possible models include partnerships with RWAs, NGOs, college volunteer cells, civic-tech organizations, or municipality pilots. The community version can remain free.

## What did you build in the hackathon?

A working MVP where users can submit a civic issue, generate an AI-powered structured report, save it, view it on a map, and copy a formal complaint message.
